# 🚛 Telegram AI Logistics Assistant Bot

## 📌 Описание проекта
Telegram-бот для анализа эффективности транспортной логистики с использованием искусственного интеллекта.

Бот принимает сообщения пользователя в Telegram и передает их ИИ-ассистенту OpenAI через API.  
Ассистент анализирует эффективность автопарка, выявляет точки потери прибыли и формирует управленческие рекомендации.

Проект создан в рамках курса по работе с AI-ассистентами и API.

---

## ⚙️ Основные возможности

- анализ эффективности автопарка
- выявление узких мест в логистике
- поиск точек потери прибыли
- рекомендации по повышению эффективности перевозок
- структурированный аналитический отчет

---

## 🧠 Используемые технологии

- **Python**
- **OpenAI API**
- **python-telegram-bot**
- **python-dotenv**
- **Git / GitHub**

---

## 🏗 Архитектура проекта

Telegram → Telegram Bot → Python Script → OpenAI Assistant → Ответ пользователю

Пользователь отправляет запрос в Telegram.  
Бот получает сообщение, передает его ИИ-ассистенту OpenAI и возвращает сформированный аналитический ответ.

## 🚀 Установка и запуск

### 1. Клонировать репозиторий

```

git clone [https://github.com/andrgol12-sys/LogAss_Bot](https://github.com/andrgol12-sys/LogAss_Bot)

```

### 2. Установить зависимости

```

pip install python-telegram-bot openai python-dotenv

```

### 3. Создать файл `.env`

Добавьте в файл следующие переменные:

```

OPENAI_API_KEY=your_openai_key
ASSISTANT_ID=your_assistant_id
TELEGRAM_TOKEN=your_telegram_token

```

### 4. Запустить бота

```

python main.py

```
